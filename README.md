# Express Error Handler Starter

Includes:
* [morgan](https://www.npmjs.com/package/morgan)
* [helmet](https://www.npmjs.com/package/helmet)
* [nodemon](https://www.npmjs.com/package/nodemon)
* [eslint](https://www.npmjs.com/package/eslint)

## Setup

```
npm install
```

## Lint

```
npm run lint
```

## Development

```
npm run dev
```
